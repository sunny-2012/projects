Nitech is a modern electronics e-commerce platform featuring products across popular brands like Samsung, Bose, JBL, Apple, LG, and more. The project delivers a seamless online shopping experience with product browsing, brand filtering, deals, and a clean UI optimized for conversions.

Built with Next.js and TypeScript, the platform uses Tailwind CSS for a fully responsive design and the Geist font for crisp, modern typography. The architecture follows Next.js App Router conventions for fast page loads and SEO-friendly server-side rendering.

Key project features:
- Product catalogue with brand filtering across Samsung, Bose, JBL, Apple, LG, Jabra, and more
- New Arrivals section with star ratings and INR pricing
- Deals and promotions section for highlighted offers
- Search functionality for quick product discovery
- User authentication with Sign In support
- Fully responsive design for desktop and mobile
- Fast, scalable performance deployed on Vercel

The result is a polished, production-ready e-commerce storefront that demonstrates scalable front-end architecture applied to real-world retail needs.
