BMW Landing Page is a high-fidelity concept website for the BMW M4, built around the brand's philosophy of "Pure Precision Engineered." The project demonstrates modern front-end development with a focus on bold visual design and a premium user experience that puts the car at center stage.

Built with React, TypeScript, Vite, and Tailwind CSS, the site features a full-screen hero section with dynamic typography, a dark cinematic aesthetic, and fast performance optimized through Vite's build pipeline.

Key project features:
- Full-screen hero with bold headline and car imagery
- Dark, high-contrast UI reflecting BMW's premium brand identity
- Responsive design optimized for all screen sizes
- React + TypeScript for type-safe component architecture
- Tailwind CSS for consistent, utility-first styling
- Smooth interactions and navigation across sections (Models, Performance, Design, Experience, Innovation)

The result is a visually striking automotive landing page that showcases how modern web technologies can be applied to luxury brand storytelling.
